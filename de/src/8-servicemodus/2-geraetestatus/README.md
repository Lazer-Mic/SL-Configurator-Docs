# Gerätestatus

![Gerätestatus](geraetestatus.png)
*Die Einschaltzeit, der Hardwarestatus und das Statusprotokoll können über das Menü Gerätestatus eingesehen werden*

**Umfassende Geräteüberwachung und Statusprotokollierung**

Detaillierte Überwachung von Gerätestatus, Betriebszeiten und Systemereignissen mit umfassender Protokollierung und Hardware-Diagnosefunktionen.

## Hauptbereiche

### 1. Einschaltzeit-Monitoring
- Kontinuierliche Überwachung der Gerätebetriebszeit
- Präzise Zeiterfassung seit dem letzten Neustart
- Akkumulierte Betriebsstunden-Statistiken
- Langzeit-Betriebsanalyse

### 2. Hardware-Status
- Echtzeit-Überwachung des Gerätezustands
- Status-Indikator mit OK/Fehler-Anzeige
- Hardware-Komponenten-Überwachung
- Systemintegritätsprüfung

### 3. Status-Protokoll
- Chronologische Ereignisprotokollierung
- Detaillierte Zeitstempel für alle Ereignisse
- Kategorisierte Ereignistypen
- Umfassende Systemhistorie

### 4. Aktualisierungsfunktion
- Manuelle Statusaktualisierung
- Echtzeit-Datenabfrage
- Sofortige Systemzustandserfassung
- Live-Monitoring-Unterstützung
